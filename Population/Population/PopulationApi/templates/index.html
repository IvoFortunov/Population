Welcome to Population Api Home page!

Apis:

getcountries: returns list of country codes

getpopulation: returns prediction for specific couuntry

http://localhost:8000/population/getpopulation/?code=USA&year=2025

http://localhost:8000/population/getlargestcountries/?year=2025&page=2
